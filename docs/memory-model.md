# Memory Model

Canonical memory is stored in `.ai/`.

- `manifest.yaml`: project identity, commands, paths, and rules.
- `state.md`: what is true now.
- `decisions.md`: durable decisions.
- `tasks.md`: active next steps and backlog.
- `tests.md`: validation matrix.
- `graph.md`: human-readable graph summary.

Canonical memory stores conclusions, not raw chat transcripts.
