# Adapters

Adapters export AMO memory into external tool formats.

Initial targets:

- `AGENTS.md`
- Claude Code `CLAUDE.md`
- Cursor rules
- Cline Memory Bank
- Codex-compatible instructions
- OpenCode instructions
- MCP resources/tools
- Obsidian notes

Adapters are not the source of truth. `.ai/` is the source of truth.
